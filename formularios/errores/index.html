Guía para documentar incidencias

Para cada error encontrado, responde las siguientes preguntas:

  • ¿Qué error encontraste? (Descripción clara del problema)
  • ¿Cómo se puede reproducir? (Pasos detallados)
  • ¿Cuál debería ser el comportamiento esperado?
  • ¿Qué se obtiene en su lugar?
  • ¿Alguna sugerencia de solución?

Ejemplo:

"El botón 'Enviar' no valida correctamente el email. Para reproducirlo, ingresa un email sin '@' y presiona enviar. Debería mostrar un mensaje de error inmediato, pero no lo hace. Posible solución: revisar la función de validación en JavaScript."
